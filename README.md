# AI 写作助手

一款类似 Cursor 的 AI 写作工具（Web 端），支持智能写作、知识库管理、文档分享等功能。

## 核心功能

### 1. AI 写作
- 根据知识库或公共文章进行文档生成
- 文档扩写、润色、翻译等功能
- 智能化的写作辅助

### 2. 知识库管理
- 个人私有知识库
- 支持文件夹和文档两级结构
- 文件夹可无限嵌套

### 3. 公共文档
- 支持文档公开分享
- 浏览其他用户的公开文档

### 4. AI 搜索与推荐
- 智能搜索私人知识库和公共文档
- 首页个性化推荐功能

## 技术栈

- **前端框架**: Vue 3
- **编辑器**: Vditor (Markdown 编辑器)
- **构建工具**: Vite
- **路由**: Vue Router

## 项目结构

```
src/
├── views/
│   └── HomePage.vue          # 首页主组件
├── components/
│   ├── SideNav.vue           # 左侧导航栏
│   └── ArticleCard.vue       # 文章卡片组件
├── App.vue                   # 根组件
└── main.js                   # 入口文件
```

## 开发指南

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产构建
```bash
npm run preview
```

## 界面特点

1. **现代化设计**: 采用简洁的卡片式布局，渐变色按钮，提升视觉体验
2. **响应式布局**: 支持不同屏幕尺寸自适应
3. **交互友好**: 悬停效果、过渡动画，提升用户体验
4. **清晰导航**: 左侧固定导航栏，便于快速切换功能
